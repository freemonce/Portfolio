# HTML과 CSS만을 활용하여 제작한 **모바일 반응형 이력서 포트폴리오**입니다.

## Live Demo
* [배포 링크 주소 입력 - 예: [GitHub Pages](https://freemonce.github.io/Portfolio/)]

## Key Features
- **Mobile-First Design**: 440px 너비의 모바일 환경에 최적화된 레이아웃 구현
- **hover 구성 강화**:  각 요소마다 시각적인 재미를 더한 요소들을 활용
- **Semantic Markup**: 웹 접근성을 고려하여 `<header>`, `<main>`, `<section>`, `<footer>` 등의 시맨틱 태그 활용
- **Icon Visualization**: Font Awesome 및 SVG 아이콘을 활용한 직관적인 UI 구성
- **Interactive Stars**: 별점 시스템을 시각적으로 구현하여 보유 기술 수준 표시
- **모바일/태블릿/데스크톱 반응형 구현**: 모바일 → 태블릿 → 데스크톱 순으로 화면을 넓히면서 `header` , `main` 의 구성요소들을 변화 조정.

## Growth & Learning (고민했던 점)
이 프로젝트를 진행하며 다음과 같은 부분에 집중했습니다:
- **시맨틱 구조의 중요성**: 단순 `<div>` 나열에서 벗어나 검색 엔진과 스크린 리더가 이해하기 쉬운 구조로 코드를 개선했습니다.
- **레이아웃 안정성**: `position: absolute`를 적절히 제어하여 요소들이 겹치지 않고 의도한 위치에 배치되도록 CSS를 세밀하게 조정했습니다.
- **이미지 최적화**: 배경 이미지와 아이콘 이미지를 CSS `object-fit` 속성으로 관리하여 해상도 대응을 학습했습니다.
